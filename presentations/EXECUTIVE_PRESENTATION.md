## Appendix: Technical Details

### 12 Factor Agents Principles

1. **Single-Purpose Agents** - Each agent has one job
2. **Explicit Dependencies** - All dependencies versioned
3. **Configuration Management** - Config in environment
4. **External Tools** - Databases as resources
5. **Deterministic Deployment** - Reproducible builds
6. **Stateless Execution** - No internal state
7. **Port Binding** - Services via ports
8. **Concurrency** - Horizontal scaling
9. **Disposability** - Fast startup/shutdown
10. **Dev/Prod Parity** - Same environments
11. **Logs as Streams** - Structured logging
12. **Admin Processes** - Separate admin tasks

*Industry best practices for AI systems*
